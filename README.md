# Netflix Clone 🎬

A modern and responsive Netflix landing page clone built using pure HTML and CSS. This project focuses on replicating the UI/UX of Netflix with attention to layout, styling, and visual details.

## 🚀 Live Demo
https://samridhii-20.github.io/netflix-clone/

---

## ✨ Features

- Clean and modern Netflix-inspired UI
- Hero section with background overlay
- Trending section with horizontal scroll
- Styled ranking cards with overlay numbers
- Interactive FAQ section layout
- Smooth hover effects and transitions
- Pixel-aligned design inspired by real Netflix

---

## 🛠️ Tech Stack

- HTML5
- CSS3 (Flexbox, Grid, Positioning, Gradients)
